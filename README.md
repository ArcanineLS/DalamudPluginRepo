## 📥 Installation

In-game, open **Dalamud Settings** (`/xlsettings`) → **Experimental** → **Custom Plugin Repositories**, and add:

```text
https://raw.githubusercontent.com/ArcanineLS/DalamudPluginRepo/master/pluginmaster.json
```

Click **+**, **Save**, then open the **Plugin Installer** (`/xlplugins`) and install any plugin below.

## ⚙️ How it works

[`pluginmaster.json`](pluginmaster.json) is generated by [`generate_pluginmaster.py`](generate_pluginmaster.py) from the per-plugin manifests under [`plugins/`](plugins). The [**Regenerate PluginMaster**](.github/workflows/regenerate.yml) workflow runs it automatically:

- on every push that touches `plugins/`,
- on a 12-hour schedule (to refresh download counts),
- and on manual dispatch.

It has no cross-repo dependencies — each plugin builds and publishes its own `latest.zip` release, and the download link is derived as `{RepoUrl}/releases/download/v{AssemblyVersion}/latest.zip`.

### Adding or updating a plugin

1. Add/update `plugins/<InternalName>/<InternalName>.json` with the plugin's manifest (`AssemblyVersion`, `RepoUrl`, etc.).
2. Push — the workflow regenerates `pluginmaster.json`.
3. Ensure the plugin repo has a matching release (tag `v{AssemblyVersion}`) with `latest.zip` attached.

Regenerate locally instead:

```bash
pip install -r requirements.txt
python generate_pluginmaster.py
```
