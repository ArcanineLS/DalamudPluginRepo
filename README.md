# DalamudPluginRepo

A custom [Dalamud](https://github.com/goatcorp/Dalamud) plugin repository for **[ArcanineLS](https://github.com/ArcanineLS)**'s FFXIV plugins.

## Installation

Add the following repo to your Dalamud custom plugin repositories
(`/xlsettings` → **Experimental** → **Custom Plugin Repositories**):

```text
https://raw.githubusercontent.com/ArcanineLS/DalamudPluginRepo/master/pluginmaster.json
```

Then open the Plugin Installer (`/xlplugins`) and install from the list below.

## How it works

`pluginmaster.json` is generated by [`generate_pluginmaster.py`](generate_pluginmaster.py) from the per-plugin manifests under [`plugins/`](plugins). The [Regenerate PluginMaster workflow](.github/workflows/regenerate.yml) runs it automatically on every push that touches `plugins/`, on a 12-hour schedule (to refresh download counts), and on manual dispatch.

### Adding or updating a plugin

1. Add/update `plugins/<InternalName>/<InternalName>.json` with the plugin's manifest (`AssemblyVersion`, `RepoUrl`, etc.).
2. Push. The workflow regenerates `pluginmaster.json`, deriving the download link as `{RepoUrl}/releases/download/v{AssemblyVersion}/latest.zip`.
3. Make sure the plugin repo has a matching release (tag `v{AssemblyVersion}`) with `latest.zip` attached.

You can also regenerate locally:

```bash
pip install -r requirements.txt
python generate_pluginmaster.py
```
